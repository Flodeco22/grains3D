detects simple pairs in a 3D binary image 

{\bfseries{Usage\+:}} simplepair in.\+pgm [all] out.\+pgm

{\bfseries{Note\+: }} in the python front-\/end the parameter \textquotesingle{}all\textquotesingle{} is mandatory, if 0 at most one simple pair is found, if it\textquotesingle{}s 1 all simple pairs are looked for

{\bfseries{Description\+:}} Detects simple pairs in a 3D binary image. If the optional parameter {\bfseries{all}} is used, then all simple pairs are searched. Otherwise, only one pair (if any) is given.
