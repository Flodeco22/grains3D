Detailed Description

interactive visualization / segmentation of a 3D image

Usage: 3dview.tcl in.pgm out.pgm [mark.pgm]
3dview in.pgm out.pgm [mark.pgm]

Description:

Displays three cross sections of a 3D grayscale image, and allows for simple segmentation by thresholding. If given, the parameter mark.pgm must be a binary image (to be superimposed with the input image in.pgm). Segmentation result is saved in out.pgm.

Menus:

  • File: choose Quit to save and exit.
  • Analyze: to compute the histogram of the full image (item Histogram) or of the Region Of Interest specified by the current segmentation
    (item ROI Histogram).
  • Threshold: simple of double thresholding.
  • Visualize: choose Revert to inverse the gray levels, check Surimp to superimpose the binary image to the grayscale data, check Light for superimposition to be made with a light color, choose Render or Rendersmooth to launch a 3D rendering of the binary image (a smoothing of the surface is done with the last choice).

Types supported: byte 3d
