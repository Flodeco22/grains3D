\doxysubsection{Detailed Description}
Noise filtering with the Ungerflow algorithm [1]. 

{\bfseries{Usage\+:}} ungerflow( (char\+\_\+image)source and sink, (float\+\_\+image)gradient image, (float\+\_\+image)lambda image, (float\+\_\+image)guidence image, (int)number of iterations, (float)tau, (float)theta) -\/$>$ float\+\_\+image

{\bfseries{Description\+:}}

The input image is noise filtered according to the following relation\+:  \[
  \min\limits_{u, v}\left\{
    \int\limits_\Omega g(x)|\nabla u|d\Omega + 
    \frac{1}{\theta}\int\limits_\Omega(u-v)^2d\Omega + 
    \int\limits_\Omega \lambda(x)|v-f|d\Omega
    \right\}
  \]

{\bfseries{Types supported\+:}} float 2d, float 3d, float 4d, ..., float xd
