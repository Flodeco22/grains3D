surfacic skeleton

Usage: in.pgm n out.pgm

Description: Surfacic skeleton in the sense defined in ref. [LB99] The parameter n gives the number of parallel steps of simple, non surface end point deletions.

Reference:
[LB99] C. Lohou, Gilles Bertrand: "Poset approach to 3D parallel thinning", SPIE Vision Geometry VIII, Vol. 3811, pp. 45-56, 1999.

Types supported: byte 3d
