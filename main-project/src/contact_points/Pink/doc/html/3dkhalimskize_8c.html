Detailed Description

converts a 3D image into a 3D Khalimsky order

Usage: 3dkhalimskize in.pgm {6|26|h|m|M|a|R|b} out.pgm

Description: Converts a 3D image into a 3D Khalimsky order.

Modes 6, 26 and h are for binary images.

Modes m, M and a are for grayscale images.

Mode =

  • 6 : emulates 6-connectivity, idem "miss",
  • 26 : emulates 26-connectivity ("hit" transform followed by a kernel transform),
  • h : "hit",
  • m : min,
  • M : max,
  • a : average,
  • R : reverse,
  • b : border.

Types supported: byte 3d, long 3d, float 3d

Category: orders
