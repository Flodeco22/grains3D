\doxysubsection{Detailed Description}
draw line segments that represent a vector field (represented by two lists) 

{\bfseries{Usage\+:}} drawfieldlists in1.\+list in2.\+list in.\+pgm [len] out.\+pgm

{\bfseries{Description\+:}} The files {\bfseries{in1.\+pgm}}, {\bfseries{in2.\+pgm}} represent a vector field, with the origins of vectors in {\bfseries{in1.\+pgm}} and their directions in {\bfseries{in2.\+pgm}}. This operator draws in the image {\bfseries{in.\+pgm}} a straight line segment of length {\bfseries{len}} (default 10) for each vector.

{\bfseries{Types supported\+:}} byte 2D, byte 3D
