\doxysubsection{Detailed Description}
parallel 2D and 3D binary guided thinning 

{\bfseries{Usage\+:}} skel\+\_\+\+CKG in.\+pgm prio.\+pgm val out.\+pgm

{\bfseries{Description\+:}} Parallel 2D and 3D binary guided thinning based on critical kernels. The parameter {\bfseries{in.\+pgm}} specifies the set (object) to be thinned. The parameter {\bfseries{prio.\+pgm}} specifies the priority function. The parameter {\bfseries{val}} is a threshold\+: any pixel having a priority greater than or equal to {\bfseries{val}} will be preserved from deletion. If {\bfseries{val}} equals -\/1, then all points will be considered for deletion.

{\bfseries{Warning\+:}} The object must not have any point on the frame of the image.

{\bfseries{Types supported\+:}} byte 2d, byte 3d
