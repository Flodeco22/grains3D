\doxysubsection{Detailed Description}
converts a binary image into a simple graph 

{\bfseries{Usage\+:}} pgm2graph in.\+pgm connex out.\+graph

{\bfseries{Description\+:}}

The vertices of the graph are the object pixels of the binary image {\bfseries{in.\+pgm}}, and there is an edge between each pair of adjacent pixels (according to {\bfseries{connex}}). ~\newline


{\bfseries{Types supported\+:}} byte 2d, byte 3d
