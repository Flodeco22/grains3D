\chapter{Pink}
\label{index}\index{Pink@{Pink}}
Pink is an image processing library developed at {\texttt{ ESIEE Engineering}} for research and teaching purposes. It contains implementations of over 200 algorithms for image segmentation and filtering. Most of the operators come from mathematical morphology, but it contains operators from different fields. It is a free software licensed under the {\texttt{ Ce\+CILL }} license.

In august 2011 Pink has been presented at the European Python Scientific Conference as a poster. Below you can see the conference poster (also in {\texttt{ pdf }} and hi-\/res {\texttt{ jpeg }}).
