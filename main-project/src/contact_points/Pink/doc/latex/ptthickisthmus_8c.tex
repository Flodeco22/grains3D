{\bfseries{Usage\+:}} ptthickisthmus in.\+pgm out.\+pgm

{\bfseries{Description\+:}} An thick end point is a white point, which has only white n-\/neighbours in the same quadrant (2d) or octant (3d), and at least one. Connectivity n is 8 (2D) or 26 (3d).
