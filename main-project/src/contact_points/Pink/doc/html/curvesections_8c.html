Detailed Description

computes a series of normal 2d cross sections from a 3d curve and image

Usage: curvesections in.pgm curve.list tolerance n_samples clip_rad out_rs out_cs out_prefix

Description: Computes a series of normal 2d cross sections from a 3d curve curve.list and image in.pgm.

Method:

1/ computes a cubic spline from the curve, with tolerance given by parameter tolerance .

2/ samples the spline to obtain n_samples points, and computes the tangents at these points

3/ at each sample point, extract a 2D section from in.pgm in the plane normal to the tangent. Delete the pixels at distance larger than clip_rad from the basis origin.

Types supported: byte 3d, int32_t 3d
