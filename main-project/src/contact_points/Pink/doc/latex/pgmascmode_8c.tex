\doxysubsection{Detailed Description}
converts a pgm file to ascii mode 

{\bfseries{Usage\+:}} pgmascmode filename [outfilename $\vert$ "{}wh"{} outfilename]

{\bfseries{Description\+:}} Converts a pgm file to ascii mode. If only parameter {\bfseries{filename}} is given, then the result is also named {\bfseries{filename}} . If the keyword "{}wh"{} is used as second parameter (stands for "{}without header"{}), then an ascii file without header is produced.

{\bfseries{Types supported\+:}} byte 2d, byte 3d, int32\+\_\+t 2d, int32\+\_\+t 3d, float 2d, float 3d
