\doxysubsection{Detailed Description}
homotopic thinning based on critical kernels 

{\bfseries{Usage\+:}} 3dskelck in.\+pgm mode niter [inhibit] out.\+pgm

{\bfseries{Description\+:}}

Homotopic thinning of a 3d Khalimsky image. The maximum number of steps is given by {\bfseries{niter}}. If the value given for {\bfseries{niter}} equals -\/1, the thinning is continued until stability.

The parameter {\bfseries{mode}} specifies the algorithm used for thinning. Possible choices are\+: \begin{DoxyItemize}
\item 0\+: ultimate thinning, symmetrical \item 1\+: ultimate crucial thinning, symmetrical \item 2\+: surface thinning, symmetrical \item 3\+: curve thinning, symmetrical \item 4\+: surface and curve thinning, symmetrical \item 5\+: surface and curve thinning (variant), symmetrical \item 6\+: curve thinning (variant), symmetrical\end{DoxyItemize}
If the parameter {\bfseries{inhibit}} is given and is a binary image name, then the points of this image will be left unchanged.

\begin{DoxyWarning}{Warning}
Input must be a complex with no point on the image border.
\end{DoxyWarning}
