\doxysubsection{Detailed Description}
This class forms the base of all image classes. 

It is used in template metaprogramming. As all the image types are derived from this type, you can test for an arbitrary type if it\textquotesingle{}s an image. See \doxyref{pink\+::python\+::convert\+\_\+if}{p.}{structpink_1_1python_1_1convert__if} .
