adds a border with a given gray value to an image 

{\bfseries{Usage\+:}} enframe3d in.\+pgm [grayvalue [width [out.\+pgm]]]

{\bfseries{Description\+:}} Adds a border to the input image, filled with the value {\bfseries{grayvalue}} (default 0). The width of the border may be given as parameter {\bfseries{width}}, otherwise its value is 1.

If {\bfseries{out.\+pgm}} is not specified, then out.\+pgm = in.\+pgm.
