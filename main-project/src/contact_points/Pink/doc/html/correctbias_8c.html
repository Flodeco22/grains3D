Detailed Description

correction of a luminosity bias expressed by a linear function

Usage: correctbias in.pgm factor type [xc yc] out.pgm

Description: Let a be the value of parameter factor. The bias type can be:

  • 0 radial (center xc, yc), linear (add -ax to the value of the element of abcissa x).
  • 1 radial (center xc, yc), parabolic (add -a^2 x to the value of the element of abcissa x).

Types supported: byte 2D
