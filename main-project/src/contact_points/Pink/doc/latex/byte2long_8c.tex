\doxysubsection{Detailed Description}
converts a "{}byte"{} image to a "{}int32\+\_\+t"{} image 

{\bfseries{Usage\+:}} byte2long in [out]

{\bfseries{Description\+:}}

For each pixel x, out[x] = (int32\+\_\+t)in[x].

If the last argument {\bfseries{out}} is omitted, then out = in.

{\bfseries{Types supported\+:}} byte 2d, byte 3d.
