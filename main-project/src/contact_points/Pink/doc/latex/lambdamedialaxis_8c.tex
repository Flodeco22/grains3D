discrete lambda-\/medial axis transform 

{\bfseries{Usage\+:}} lambdamedialaxis in.\+pgm [lambda] out.\+pgm

{\bfseries{Description\+:}} Discrete lambda-\/medial axis, as defined in [CCT09], of the binary image {\bfseries{X}} contained in {\bfseries{in.\+pgm}}.

By default, the output is a float image representing the function x-\/$>$lambda(x).

If the parameter {\bfseries{lambda}} is given (bash interface only, not python), the output is a binary image obtained by thresholding the aforementioned function at the value {\bfseries{lambda}} .

{\bfseries{Usage\+:}} lambdamedialaxis in.\+pgm [lambda] out.\+pgm

{\bfseries{Description\+:}} Discrete lambda-\/medial axis, as defined in [CCT09], of the binary image {\bfseries{X}} contained in {\bfseries{in.\+pgm}}.

By default, the output is a float image representing the function x-\/$>$lambda(x).

If the parameter {\bfseries{lambda}} is given (bash interface only, not python), the output is a binary image obtained by thresholding the aforementioned function at the value {\bfseries{lambda}} .

If the parameter {\bfseries{lambda}} is given and set to 0, the output is a binary image obtained by thresholding the aforementioned function at the automaticaly defined (k-\/means) value {\bfseries{lambda}} .
