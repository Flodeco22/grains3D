draw spline orientations in a vector field (multiple spline version) 

{\bfseries{Usage\+:}} drawsplinesorient in.\+pgm in.\+splines out.\+pgm

{\bfseries{Description\+:}} Draws spline orientations in a vector field. The spline is specified by its control points in a text file. The parameter {\bfseries{in.\+pgm}} gives a vector field into which the spline is to be drawn.

The file {\bfseries{splines.\+txt}} contains a list of splines under the format\+:~\newline
 d nb\+\_\+splines~\newline
 nb\+\_\+points\+\_\+spline\+\_\+1 x11 y11 x12 y12 ...~\newline
 nb\+\_\+points\+\_\+spline\+\_\+2 x21 y21 x22 y22 ...~\newline
 nb\+\_\+points\+\_\+spline\+\_\+3 x31 y31 x32 y32 ...~\newline
 ...~\newline
 or, in 3D\+:~\newline
 D nb\+\_\+splines~\newline
 nb\+\_\+points\+\_\+spline\+\_\+1 x11 y11 z11 x12 y12 z12 ...~\newline
 nb\+\_\+points\+\_\+spline\+\_\+2 x21 y21 z21 x22 y22 z22 ...~\newline
 nb\+\_\+points\+\_\+spline\+\_\+3 x31 y31 z31 x32 y32 z32 ...~\newline
 ...~\newline


{\bfseries{Types supported\+:}} spline 2D, spline 3D
