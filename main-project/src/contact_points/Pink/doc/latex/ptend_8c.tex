\doxysubsection{Detailed Description}
detects end points in a binary image 

{\bfseries{Usage\+:}} ptend in.\+pgm connex out.\+pgm

{\bfseries{Description\+:}} An end point is a white point, which has exactly 1 white n-\/neighbour (n = 4, 8 (2d) or 6, 18, 26 (3d), as set by the parameter {\bfseries{connex}})

When the type of {\bfseries{in.\+pgm}} is 4\+\_\+\+BYTE, the image is treated as a label image, where each label is processed as a separate binary image (all other labels are considered as background).

{\bfseries{Types supported\+:}} byte 2D, byte 3D, long 3D
