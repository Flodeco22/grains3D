Detailed Description

opening function

Usage: openingfunction in.pgm [dist] out.pgm

Description: This operator associates, to each point x of the input object X, the radius of the biggest ball included in X that includes x. The distance used depends on the optional parameter dist (default is 0) :

  • 0: Euclidean distance (truncated)
  • 1: approximate quadratic Euclidean distance
  • 2: chamfer distance
  • 3: exact quadratic Euclidean distance
  • 4: 4-distance in 2d
  • 8: 8-distance in 2d
  • 6: 6-distance in 3d
  • 18: 18-distance in 3d
  • 26: 26-distance in 3d
Warning
The input image in.pgm must be a binary image. No test is done.

Types supported: byte 2D, byte 3D
