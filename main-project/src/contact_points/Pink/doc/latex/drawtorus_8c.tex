\doxysubsection{Detailed Description}
generates a binary torus 

{\bfseries{Usage\+:}} drawtorus \{in.\+pgm$\vert$null\} r1 r2 xc yc zc out.\+pgm

{\bfseries{Description\+:}} Draws a binary torus in the image {\bfseries{in.\+pgm}}. The parameters {\bfseries{r1, r2}} specify respectively the big and the small radius of the torus. The parameters {\bfseries{xc, yc, zc}} specify the center of the torus. If the first parameter is "{}null"{}, then an image with the appropriate size is created.

{\bfseries{Types supported\+:}} byte 3d
