suppress the header from a pgm file

Usage: pgm2raw in.pgm out.mhd out.raw

Description: converts a pgm file to mhd format Types supported: byte 2d, byte 3d, short 2d, short 3d, long 2d, long 3d, float 2d, float 3d, double 2d, double 3d

Category: convert

Usage: pgm2raw image_name[.pgm] [out.raw]

Description: suppress the header from a pgm file. If used with only one argument image_name, the operator searches for a file named image_name.pgm and generates a raw file named image_name.raw, a NRRD header file named image_name.nrrd (see http://teem.sourceforge.net/nrrd/format.html), and a MHD header file named image_name.mfd (see https://itk.org/Wiki/ITK/MetaIO/Documentation)

Types supported: byte 2d, byte 3d, short 2d, short 3d, long 2d, long 3d, float 2d, float 3d, double 2d, double 3d
