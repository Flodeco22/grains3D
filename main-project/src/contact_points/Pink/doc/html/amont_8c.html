Detailed Description

amont of a binary marker in a grayscale image

Usage: amont in.pgm mark.pgm connex [s] out.pgm

Description: Let F be the function defined by in.pgm, let X be the set defined by mark.pgm. The amont of X in F is defined as the set of points x such that there exists an ascending path from X to x for F with connectivity connex. If the parameter s is given, then only strictly ascending paths are considered.

Types supported: byte 2d
