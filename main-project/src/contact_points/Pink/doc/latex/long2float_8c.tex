\doxysubsection{Detailed Description}
converts a "{}long"{} image to a "{}float"{} image 

{\bfseries{Usage\+:}} long2float in [out]

{\bfseries{Description\+:}} For each pixel x, out[x] = (float)in[x]

If the last argument {\bfseries{out}} is omitted, then out = in.

{\bfseries{Types supported\+:}} long 2d, long 3d.
