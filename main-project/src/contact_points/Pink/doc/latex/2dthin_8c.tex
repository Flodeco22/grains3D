homotopic thinning of a 2d Khalimsky image 

{\bfseries{Usage\+:}} 2dthin in.\+pgm n out.\+pgm

{\bfseries{Description\+:}} Homotopic thinning of a 2d Khalimsky image. The maximum number of steps is given by {\bfseries{n}}. If the value given for {\bfseries{n}} equals -\/1, the thinning is continued until stability.

{\bfseries{Types supported\+:}} byte 2d
