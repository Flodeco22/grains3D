Detailed Description

ultimate erosion

Usage: ultimateerosion in.pgm [dist] out.pgm

Description: Let X be the set in in.pgm . The result is union{Ui(X), i in N} where Ui(X) = erosball(X,i) \ reconsgeo(erosball(X,i+1), erosball(X,i)). Structuring elements are balls defined after a distance. The distance used depends on the optional parameter dist (default is 0) :

  • 0: approximate euclidean distance (truncated)
  • 1: approximate quadratic euclidean distance
  • 2: chamfer distance
  • 3: exact quadratic euclidean distance
  • 4: 4-distance in 2d
  • 8: 8-distance in 2d
  • 6: 6-distance in 3d
  • 18: 18-distance in 3d
  • 26: 26-distance in 3d
Warning
The input image in.pgm must be a binary image. No test is done.

Types supported: byte 2D, byte 3D
