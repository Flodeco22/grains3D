\doxysubsection{Detailed Description}
draw spline segments which are specified by a text file 

{\bfseries{Usage\+:}} drawsplines in.\+pgm splines.\+txt [len] out.\+pgm

{\bfseries{Description\+:}} Draws splines which are specified by control points in a text file. The parameter {\bfseries{in.\+pgm}} gives an image into which the splines are to be drawn. The file format for {\bfseries{splines.\+txt}} is the following for 2D\+:

The file {\bfseries{splines.\+txt}} contains a list of splines under the format\+:~\newline
 d nb\+\_\+splines~\newline
 nb\+\_\+points\+\_\+spline\+\_\+1 x11 y11 x12 y12 ...~\newline
 nb\+\_\+points\+\_\+spline\+\_\+2 x21 y21 x22 y22 ...~\newline
 nb\+\_\+points\+\_\+spline\+\_\+3 x31 y31 x32 y32 ...~\newline
 ...~\newline
 or, in 3D\+:~\newline
 D nb\+\_\+splines~\newline
 nb\+\_\+points\+\_\+spline\+\_\+1 x11 y11 z11 x12 y12 z12 ...~\newline
 nb\+\_\+points\+\_\+spline\+\_\+2 x21 y21 z21 x22 y22 z22 ...~\newline
 nb\+\_\+points\+\_\+spline\+\_\+3 x31 y31 z31 x32 y32 z32 ...~\newline
 ...~\newline


If parameter {\bfseries{len}} is given and non-\/zero, the splines are extended on both sides by straight line segments of length {\bfseries{len}}.

{\bfseries{Types supported\+:}} byte 2D, byte 3D
