\doxysubsection{Detailed Description}
generates a binary cylinder 

{\bfseries{Usage\+:}} drawcylinder \{in.\+pgm$\vert$null\} r1 r2 xc yc zc out.\+pgm

{\bfseries{Description\+:}} Draws a binary cylinder in the image {\bfseries{in.\+pgm}}. The parameters {\bfseries{r1, r2}} specify respectively the big and the small radius of the cylinder. The parameters {\bfseries{xc, yc, zc}} specify the center of the cylinder. If the first parameter is "{}null"{}, then an image with the appropriate size is created.

{\bfseries{Types supported\+:}} byte 3d
