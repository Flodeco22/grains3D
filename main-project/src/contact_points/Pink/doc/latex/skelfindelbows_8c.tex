\doxysubsection{Detailed Description}
detection of "{}elbows"{} in a curvilinear skeleton 

{\bfseries{Usage\+:}} skelfindelbows in.\+skel thickness sharp out.\+pgm

{\bfseries{Description\+:}}

Find "{}elbows"{} (points making sharp angles) in the curves of skeleton {\bfseries{in.\+skel}}. Matching points are written as voxels in the returned image {\bfseries{out.\+pgm}}.

\begin{DoxyVerb}Let <S[0], ... S[n-1]> be the points of a cover of the curve C by digital straight line segments (DSSs). 
Let j be an index between 1 and n-2, if angle(S[j-1]S[j], S[j]S[j+1]) <= sharp then output S[j].
\end{DoxyVerb}


Parameter {\bfseries{thick}} (thickness for DSS recognition) is given in pixels, parameter {\bfseries{sharp}} in degrees.

{\bfseries{Types supported\+:}} skel 2d, skel 3d
