Detailed Description

parallel 3D binary curvilinear, asymetric skeleton based on 3D and 2D residuals

Usage: skel_ACK3 in.pgm nsteps [inhibit] out.pgm

Description: Parallel 3D binary thinning or curvilinear, asymetric skeleton based on 3D and 2D residuals. The parameter nsteps gives, if positive, the number of parallel thinning steps to be processed. If the value given for nsteps equals -1, the thinning is continued until stability.

If the parameter inhibit is given and is a binary image name, then the points of this image will be left unchanged.

Warning: The object must not have any point on the frame of the image.

Types supported: byte 3d

Category: topobin
