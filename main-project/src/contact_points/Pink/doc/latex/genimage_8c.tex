\doxysubsection{Detailed Description}
generates a monocolor image of given size 

{\bfseries{Usage\+:}} genimage \{in.\+pgm$\vert$rs cs ds\} col out.\+pgm

{\bfseries{Description\+:}} The size of the result image {\bfseries{out.\+pgm}} is taken from image {\bfseries{in.\+pgm}}, or given by the parameters {\bfseries{rs}}, {\bfseries{cs}}, {\bfseries{ds}}. For a 2D image, the parameter {\bfseries{ds}} (depth size) is set to 1. The color of the result image is given by the parameter {\bfseries{col}}.

{\bfseries{Types supported\+:}} byte 2d, byte 3d
