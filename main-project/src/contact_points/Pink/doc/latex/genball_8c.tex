\doxysubsection{Detailed Description}
generates a binary euclidean ball 

{\bfseries{Usage\+:}} genball radius [dim] out.\+pgm

{\bfseries{Description\+:}} Draws a binary euclidean ball in the new image {\bfseries{out.\+pgm}}. The parameter {\bfseries{radius}} specifies the radius of the ball. The parameter {\bfseries{dim}} specifies the dimension\+: 2 (default) or 3. The center of the ball is designated as the origin (for use as structuring element).

{\bfseries{Types supported\+:}} byte 2d, byte 3d
