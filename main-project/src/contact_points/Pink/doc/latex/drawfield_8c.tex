\doxysubsection{Detailed Description}
draw line segments that represent a vector field 

{\bfseries{Usage\+:}} drawfield in.\+pgm [len] out.\+pgm

{\bfseries{Description\+:}} The file {\bfseries{in.\+pgm}} contains a vector field. This operator generates a (binary) image where each vector is represented by a straight line segment of length {\bfseries{len}} (default 10).

{\bfseries{Types supported\+:}} byte 2D, byte 3D
