\doxysubsection{Detailed Description}
converts a "{}short"{} image to a "{}float"{} image 

{\bfseries{Usage\+:}} short2float in [out]

{\bfseries{Description\+:}}

For each pixel x, out[x] = (float)in[x]

If the last argument {\bfseries{out}} is omitted, then out = in.

{\bfseries{Types supported\+:}} short 2d, short 3d.
