\doxysubsection{Detailed Description}
pixelwise multiplication of two images 

{\bfseries{Usage\+:}} mult in1.\+pgm in2.\+pgm out.\+pgm

{\bfseries{Description\+:}} For each pixel x, out[x] = in1[x] $\ast$ in2[x]. If both images are byte images, and if out[x] exceeds 255, then out[x] is set to 255. Images must be of the same type and same dimensions.

{\bfseries{Types supported\+:}} byte 2d, byte 3d, int32\+\_\+t 2d, int32\+\_\+t 3d, float 2d, float 3d
