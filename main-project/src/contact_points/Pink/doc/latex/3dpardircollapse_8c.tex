\doxysubsection{Detailed Description}
parallel directional constrained collapse with surface detection and preservation 

{\bfseries{Usage\+:}} 3dpardircollapse in.\+pgm mode nsteps [inhibit] out.\+pgm

{\bfseries{Description\+:}} Parallel directional constrained collapse with skeletal elements detection and preservation.

The parameter {\bfseries{mode}} selects the strategy used to detect skeletal elements. The possible choices are\+: \begin{DoxyItemize}
\item 0\+: no element \item 1\+: any 1-\/dimensional facet \item 2\+: any 2-\/dimensional facet \item 3\+: any 2-\/dimensional or 1-\/dimensional facet\end{DoxyItemize}
The parameter {\bfseries{nsteps}} gives the number of "{}layers"{} to be removes, if the value is -\/1 then the interations continue until stability.

If the parameter {\bfseries{inhibit}} is given and is a binary image name, then the points of this image will be left unchanged.

{\bfseries{Types supported\+:}} byte 3d
