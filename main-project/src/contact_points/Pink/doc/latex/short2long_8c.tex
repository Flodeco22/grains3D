\doxysubsection{Detailed Description}
converts a "{}short"{} image to a "{}int32\+\_\+t"{} image 

{\bfseries{Usage\+:}} short2long in [out]

{\bfseries{Description\+:}}

For each pixel x, out[x] = (int32\+\_\+t)in[x].

If the last argument {\bfseries{out}} is omitted, then out = in.

{\bfseries{Types supported\+:}} short 2d, short 3d.
