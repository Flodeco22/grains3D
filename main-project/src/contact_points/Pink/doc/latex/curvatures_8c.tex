\doxysubsection{Detailed Description}
compute the curvatures of a natural cubic spline 

{\bfseries{Usage\+:}} curvatures spline.\+txt n out.\+txt

{\bfseries{Description\+:}} Samples the spline in {\bfseries{spline.\+txt}} and compute the curvature at each sample point. The file {\bfseries{spline.\+txt}} contains a 2D or 3D spline under the format\+:~\newline
 s nbpoints~\newline
 ~\newline
 x1 y1~\newline
 x2 y2~\newline
 x3 y3~\newline
 ...~\newline
 or\+: S nbpoints~\newline
 ~\newline
 x1 y1 z1~\newline
 x2 y2 z2~\newline
 x3 y3 z3~\newline
 ...~\newline


{\bfseries{Types supported\+:}} 2D, 3D splines
