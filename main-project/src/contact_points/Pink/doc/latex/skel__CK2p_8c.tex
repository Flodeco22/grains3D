parallel 2D binary curvilinear skeleton based on 1D isthmuses 

{\bfseries{Usage\+:}} skel\+\_\+\+CK2p in.\+pgm pers [inhibit] out.\+pgm

{\bfseries{Description\+:}} Parallel 2D binary thinning or curvilinear skeleton based on 1D isthmuses, with a persistence parameter\+: {\bfseries{pers}}.

If the parameter {\bfseries{inhibit}} is given and is a binary image name, then the points of this image will be left unchanged.

{\bfseries{Warning\+:}} The object must not have any point on the frame of the image.

{\bfseries{Types supported\+:}} byte 2d
