\doxysubsection{Detailed Description}
grayscale homotopic thickening 

{\bfseries{Usage\+:}} hthick in.\+pgm \{imcond.\+pgm$\vert$null\} connex niter out.\+pgm

{\bfseries{Description\+:}} Grayscale homotopic thickening (refs. [BEC97, CBB01]). The parameter {\bfseries{connex}} gives the connectivity used for the minima; possible choices are 4, 8 in 2D and 6, 26 in 3D. Let F be the function corresponding to the input image {\bfseries{in.\+pgm}}. Let G be the function corresponding to the input image {\bfseries{imcond.\+pgm}}, or the blanck function (constant 255) if the keyword {\bfseries{null}} is used. The algorithm is the following\+:

\begin{DoxyVerb}Repeat niter times:
    L = {(p,d) such that p is constructible for F and d = delta+(p,F)}
    While L not empty
       extract a couple (p,d) from L
       F(p) = min{ d, delta+(p,F), G(p) }    
Result: F
\end{DoxyVerb}


If {\bfseries{niter}} is set to -\/1, then the operator iterates until stability.

 [CBB01] M. Couprie, F.\+N. Bezerra, Gilles Bertrand\+: {\texttt{ "{}\+Topological operators for grayscale image processing"{}}}, {\itshape  Journal of Electronic Imaging}, Vol.~10, No.~4, pp.~1003-\/1015, 2001.
