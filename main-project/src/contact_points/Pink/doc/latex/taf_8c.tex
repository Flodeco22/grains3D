\doxysubsection{Detailed Description}
topological alternating filter 

{\bfseries{Usage\+:}} taf in.\+pgm connexmin r [lambdap [lambdaw]] out.\+pgm

{\bfseries{Description\+:}} Topological alternating filter. Performs alternatively topological upper filter (tuf) and topological lower filter (tlf) with increasing radius until radius {\bfseries{r}}.

If given, parameter {\bfseries{lambdap}} (resp. {\bfseries{lambdaw}}) is an integer which indicates that peaks (resp. wells) of height greater than this value must be preserved.

{\bfseries{Types supported\+:}} byte 2d
